# Create an `npi` vector

This creates an integer vector of valid NPIs.

## Usage

``` r
npi(x = integer())

is_npi(x)
```

## Arguments

- x:

  - For `npi()`: An integer vector.

  - For `is_npi()`: An object to test.

## Value

An S3 vector of class `npi`

## Examples

``` r
x <- generate(10)
x
#>  [1] 1515782287 1141270894 1937499387 1453066273 1030620980 1731918606
#>  [7] 1493897644 1805243995 1435404121 1946059061
is_npi(x)
#> [1] FALSE

x <- npi(x)
x
#> <npi[10]>
#>  [1] 1515782287 1141270894 1937499387 1453066273 1030620980 1731918606
#>  [7] 1493897644 1805243995 1435404121 1946059061
is_npi(x)
```
