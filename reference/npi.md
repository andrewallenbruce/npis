# Create an `npi` vector

This creates an integer vector of valid NPIs.

## Usage

``` r
npi(x = integer())

is_npi(x)
```

## Arguments

- x:

  - For `npi()`: An integer vector.

  - For `is_npi()`: An object to test.

## Value

An S3 vector of class `npi`

## Examples

``` r
x <- generate(10)
x
#>  [1] 1506436951 1578228714 1127089493 1749938745 1306627303 1062098073
#>  [7] 1191860649 1389764480 1524399543 1540412194
is_npi(x)
#> [1] FALSE

x <- npi(x)
x
#> <npi[10]>
#>  [1] 1506436951 1578228714 1127089493 1749938745 1306627303 1062098073
#>  [7] 1191860649 1389764480 1524399543 1540412194
is_npi(x)
```
